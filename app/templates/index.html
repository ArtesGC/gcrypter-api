Intro

GCrypter-API
It's a web version of the desktop program GCrypter

- This API offers you options and a simple environment
to you as an end user to be able to ENCRYPT your text or message easily.
- And an option for DECRYPT the same data.

After the encryption will be returned to you a json response
with the data ENCRYPTED or DECRYPTED, as is shown in the boxes bellow.

A simple overview of the GCrypter-desktop:
