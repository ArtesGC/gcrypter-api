Intro

GCrypter-API
It's a web version of the desktop program GCrypter

- This API offers you options and a simple environment
to you as an end user be able to ENCRYPT your text or message easily.
- And an option for DECRYPT the same data.

After the encryption will be returned to you a json response
with the data ENCRYPTED or DECRYPTED, as is shown in the boxes bellow.

Encrypting some text:

URL: /encryption/?text=[message_goes_here] 
// GET: /encryption/?text=amor

Decrypting some text:

URL: /decryption/?value1=[first_value_goes_here]&value2=[second_value_goes_here] 
// GET: /decryption/?value1=1266033467&value2=706432073

{
  "decrypted": "amor"
}

URL: /decryption/?values=[both_values_go_here_as_tuple] 
// GET: /decryption/?values=(1266033467,706432073)

{
  "decrypted": "amor"
}
